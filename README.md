
# Minevolution
*Minecraft server plugin based on the old Minecraft map [minevolution](https://www.curseforge.com/minecraft/worlds/minevolution) where every player is in the same world, but using packets, will see different upgrades around them. Players will mine the infinity block in the center to gain higher-tier resources to upgrade their tools in the forge and the stations around them to access new areas and higher-level tools.*

**Minecraft Version: 1.21.1**

Project Overview
-
 - **Custom Packet-Based Mining Animation**
 - **Block Tier System** *(30 Blocks all with thier own custom materials and health)*
 - **Pickaxe Enchanting** *(9 Custom Enchants to improve your mining)*
 - **Forge** *(11 Custom Recipes to forge)*
 - **Custom Npcs**
